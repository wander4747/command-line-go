# Busca IP e server

## 💻 Projeto

Um projeto simples que busca o nome do servidor e IP de um domínio X: 

## 🧑‍💻 Buscar Server Name

```sh
go run main.go servers --host amazon.com.br
```

## 🧑‍💻 Buscar IP

```sh
go run main.go ip --host amazon.com.br
```
